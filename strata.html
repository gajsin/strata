Голосовое управление

Запись голосовых сообщений, транскрипция через OpenAI Whisper, автоматическое распознавание команд из естественной речи. Скажи «потратил 500 на кофе» — ИИ всё сделает сам.

🧠

AI-исполнитель команд

OpenAI API не просто отвечает — выполняет действия. Парсинг 15+ специальных команд: создание задач, запись расходов, отметка привычек, сохранение заметок.

📋

Умные задачи

Парсинг времени из русского языка: «завтра в 9», «через 30 минут», «в следующую пятницу». Приоритеты, категории, напоминания и автоматическое завершение.

💰

Финансовый трекер

Учёт доходов и расходов через чат. Категоризация, поиск последних транзакций по описанию, корректировка сумм. Графики и аналитика через Recharts.

✨

Привычки с Fuzzy-поиском

Трекер привычек с целями и стриками. «Попил воды» → находит привычку «Пить 2л воды» по корням слов. Утро/день/вечер, визуальный прогресс.

📝

База знаний

Персональная память ИИ: заметки с тегами, поиском и закреплением. Команда APPEND_NOTE безопасно дописывает текст без потери данных.
